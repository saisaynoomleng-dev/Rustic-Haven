# Rustic Haven

> Rustic Haven is a full-stack property rental platform for a mixed-use complex located in the underwater kingdom of Atlantica. The system allows tenants to browse and rent residential units while businesses can lease office and commercial spaces.

> The application provides real-time availability, secure authentication, online payments, and an admin-managed CMS for dynamic content and listings.

> Built with a modern production-grade stack focused on performance, scalability, and developer experience.

---

## Tech Stack

- **Frontend**: Next.js 15, TypeScript, TailwindCSS, shadcn/ui
- **Backend**: Next.js Server Actions / Route Handlers
- **Database**: PostgreSQL (Neon) + Drizzle ORM
- **Auth**: Clerk
- **Payments**: Stripe
- **CMS**: Sanity
- **Caching**: Redis
- **Deployment**: Vercel
